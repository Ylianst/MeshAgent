# Test Scripts

This directory contains example test scripts and proof-of-concept code used during development.

## Contents

### buildTests/
Build and upgrade testing scripts:
- **upgradeTest/** - Contains `test-upgrade-monitor.sh` for monitoring upgrade processes

### RE_Tests/
Reverse engineering tests and proof-of-concept implementations for understanding MeshAgent's module embedding system. See [RE_Tests/README.md](RE_Tests/README.md) for detailed information.

## Purpose

These scripts are primarily used for:
- Development and testing workflows
- Understanding MeshAgent internals
- Proof-of-concept implementations
- Build verification

## Note

These are example scripts for development purposes, not production code.
